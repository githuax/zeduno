 Use These Scripts

  - Set an existing tenant admin’s password:
      - cd backend
      - npm run set:tenant-admin:password -- testadmin1@mail.com "Pass@123"
  - Set a superadmin’s password (by email):
      - cd backend
      - npm run set:superadmin:password -- testadmin1@mail.com "Pass@123"
  - Create a tenant admin (if the user doesn’t exist yet):
      - cd backend
      - npm run create:tenant-admin -- testadmin1@mail.com <tenant-slug-or-id> "Pass@123"
